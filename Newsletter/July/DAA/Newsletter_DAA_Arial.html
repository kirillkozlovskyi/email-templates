Use AI Link to Bridge the Gap Between Business Intelligence & Artificial Intelligence

AI Link is a python connector that allows data personas to collaborate and apply the power of the semantic layer to challenges in Data and Applied AI:

  • Persist data in the data warehouse; maintaining compliance standards and eliminating extracts or copies of data traditionally used for data or ML modeling
  • Publish and serve ML features; select, create, transform features needed for performant ML models using the same data and metric definitions across all your data teams
  • Bridge BI and AI; append existing data models with ML model predictions and metadata to provide a full spectrum view of analytics across your business

We have some great new updates in our latest release

Pre-built integrations for Dataiku, H2O, DataRobot, and Sagemaker - seamlessly integrated with each of these platforms for ML training and MLOps. The best news is that AI Link can work with any preferred python tool of choice that can accept dataframe input. That includes, but is not limited to:

New Features in Alpha

Principal Component Analysis (PCA) - reduce dimensionality of a dataset while retaining the most pertinent information. This allows data scientists and data engineers to work with large data sets in a notebook environment. Explore, select, and create features based on source data, not extracts.

ML Writeback - transform ML predictions and other associated metadata into a dataframe to be consumed by AtScale data models and thus BI tools. Data scientists can present their model output in the same platform (PowerBI, Tableau, Excel, etc.) as defined by the business.

Exploratory Data Analysis (EDA) Functions - core functions needed to effectively explore, tailor, and manage data and features for ML training.
